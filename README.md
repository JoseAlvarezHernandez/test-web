# commits

When its a new feature you will commit like this 

```sh
git commit -m "feature(module): what did you do"
```

When you are fixing 

```sh
git commit -m "fix(module): what did you do"
```

When you are refactoring

```sh
git commit -m "refactor(module): what did you do"
```
